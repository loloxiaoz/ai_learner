# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

这是一个AI与大模型系统学习教程项目，面向面试准备与技术晋升答辩，共10章内容。

## 项目结构

```
ai_learner/
├── content/                       # 课程内容（Markdown）
│   └── XX_章节名称/               # 各章节文件夹（数字编号）
│       ├── 01_正文.md             # 章节正文
│       ├── 02_学习检验.md         # 学习检验（10+经典问题含详细答案）
│       └── images/               # 章节配图
├── web/                           # Next.js 学习网站
│   ├── src/
│   │   ├── app/                  # App Router 页面
│   │   ├── components/           # React 组件
│   │   └── lib/                  # 工具函数（读取内容等）
│   └── public/images/            # 网站静态图片资源
├── SPEC.md                        # 产品需求文档
├── CLAUDE.md                      # 本文件
└── LICENSE                        # CC BY-SA 4.0 许可证
```

## 内容编写规范

### 正文文件（01_正文.md）
- 面向初学者和有一定基础的读者
- 包含：学习目标、核心概念、原理解释、对比表格、示意图
- 语言通俗易懂，配合代码块和图表说明
- 提供延伸阅读资源（论文、教程链接）

### 学习检验文件（02_学习检验.md）
- 包含10个以上经典面试/答辩问题
- 每个问题配有详细参考答案
- 答案包含要点拆解、对比表格、加分点提示
- 提供自测评分标准和面试技巧

## 语言要求

所有回复和文档内容使用中文。
